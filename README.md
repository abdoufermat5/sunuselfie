# sunuselfie

## Start

**Install dependencies**

```bash
pip install streamlit git+https://github.com/vladkens/twscrape.git opencv-python deepface 
```

**Run the app**

```bash
streamlit run app.py
```

You should log in with your twitter account to use the app.
